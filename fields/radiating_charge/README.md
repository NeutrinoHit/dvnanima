# Radiating Charge via Lienard-Wiechert Fields

This module visualizes electromagnetic fields generated by a moving point charge.

Model separation is strict:
- trajectory solver defines `r(t), v(t), a(t)`;
- field solver uses only Lienard-Wiechert formulas and retarded time;
- renderer consumes precomputed dataset only.

## Trajectory backends

`trajectory.model = "coulomb"`
- solves particle motion in external Coulomb field of a heavy fixed center.
- external field is used only to compute trajectory; it is not rendered.

`trajectory.model = "oscillator"`
- solves damped/undamped harmonic oscillator trajectory.

## Dataset export

```bash
python fields/radiating_charge/export_radiating_charge_dataset.py \
  --config fields/radiating_charge/radiating_charge.toml \
  --out fields/radiating_charge/radiating_charge_dataset.npz
```

Preview export:

```bash
python fields/radiating_charge/export_radiating_charge_dataset.py \
  --config fields/radiating_charge/radiating_charge.toml \
  --out fields/radiating_charge/radiating_charge_dataset_preview.npz \
  --preview
```

## Renderers

pyqtgraph (main):

```bash
python fields/radiating_charge/render_radiating_charge_pyqtgraph.py \
  fields/radiating_charge/radiating_charge_dataset.npz \
  --mode 2d --show-center --lower-key bz_rad --transform signed_log
```

```bash
python fields/radiating_charge/render_radiating_charge_pyqtgraph.py \
  fields/radiating_charge/radiating_charge_dataset.npz \
  --mode 3d --show-center --show-upper --stacked-planes --upper-key ez_rad \
  --transform signed_log --surface-alpha 0.40
```

Plotly (validation):

```bash
python fields/radiating_charge/render_radiating_charge_plotly.py \
  fields/radiating_charge/radiating_charge_dataset.npz \
  --mode 2d --show-center --lower-key bz_rad --transform signed_log --out fields/radiating_charge/radiating_charge_plotly_2d.html
```

## Recommended observables

- `e_mag` for total electric field magnitude
- `bz_rad` for signed radiation wavefronts (positive/negative lobes)
- `s_mag` for total energy flux magnitude
- `s_rad_mag` for radiation-only flux magnitude (best to see emitted wave)
